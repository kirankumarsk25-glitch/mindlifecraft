# Mind Life Craft - Static Website

This is a static website for Mind Life Craft, built with Next.js. The content is extracted from the provided PDF and presented in a clean, responsive layout.

## Overview

The website showcases the company's vision, mission, and various domains including:
- Research & Development
- Clinical Excellence (Musculoskeletal Wellness, Infertility & Progeny, Neuromuscular Disorders, Shalya Tantra)
- Coaching and Counselling
- Pharmaceutical and Wellness Products

## Building the Static Site

The project is configured for static export. To build:

```bash
npm run build
```

The static files will be generated in the `out/` directory.

## Deploying

You can deploy the contents of the `out/` folder to any static hosting service such as:
- Vercel
- Netlify
- GitHub Pages
- AWS S3 + CloudFront

## Development

To run in development mode:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Technologies Used

- Next.js 16
- TypeScript
- Tailwind CSS
- ESLint
